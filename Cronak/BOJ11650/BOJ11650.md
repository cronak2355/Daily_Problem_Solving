# BOJ 11650 - 좌표 정렬하기 문제 풀이 설명 2026/02/11

## 문제 
2차원 평면 위의 점 N개가 주어진다. 좌표를 x좌표가 증가하는 순으로, x좌표가 같으면 y좌표가 증가하는 순서로 정렬하여 출력하는 프로그램을 작성하시오.

- **입력:** 첫째 줄에 점의 개수 N ($1 \le N \le 100,000$), 둘째 줄부터 N개의 줄에 점의 위치 $x_i, y_i$가 주어진다. (-100,000 $\le x_i, y_i \le$ 100,000)
- **출력:** 정렬된 결과를 한 줄에 하나씩 출력한다.
- **조건:** 시간 제한 1초, 메모리 제한 256MB

## 풀이
이 문제는 **다중 조건 정렬(x 우선, y 차선)** 을 구현하는 것이 핵심이다.

### 1. 자료구조 선택
- 좌표 (x, y)를 묶어서 관리하기 위해 C++ STL의 `pair<int, int>`를 사용한다.
- 데이터의 개수가 정해져 있지 않고 동적으로 들어오므로 `vector`를 사용한다.

### 2. 알고리즘 (Sorting)
- `std::pair`는 비교 연산자(`<`) 사용 시 **첫 번째 인자(`first`)를 기준으로 먼저 비교하고, 값이 같으면 두 번째 인자(`second`)를 비교**하는 특성이 있다.
- 문제에서 요구하는 정렬 기준(x 오름차순 -> y 오름차순)과 `pair`의 기본 정렬 기준이 정확히 일치하므로, 별도의 `compare` 함수를 작성할 필요 없이 `sort()`만 호출하면 된다.

## 기타

### 1. 시간 복잡도 분석
- **문제의 제약:** 점의 개수 N은 최대 100,000개이다.
- **알고리즘 선택:** $O(N^2)$인 버블 정렬 등을 사용하면 연산 횟수가 약 100억 회가 되어 시간 제한(1초)을 초과한다. 따라서 평균 $O(N \log N)$을 보장하는 `std::sort`를 사용해야 한다.

### 2. Vector
- `vector<pair<int, int>> points(N);`
    - 크기가 N인 벡터를 생성하고, 모든 요소를 기본값(`{0, 0}`)으로 초기화한다.
    - 배열처럼 `points[i]` 인덱스로 접근하여 값을 넣을 수 있다.
- `vector<pair<int, int>> points;`
    - 크기가 0인 빈 벡터를 생성한다.
    - 인덱스로 접근할 수 없으며, `points.push_back({x, y})`를 사용하여 요소를 뒤에 추가해야 한다.
    - 요소의 개수를 미리 알고 있다면 `(N)`으로 할당하는 것이 메모리 재할당 비용을 줄일 수 있어 효율적이다.
