# BOJ 1074 - Z 문제 풀이 설명 2026/02/11

## 문제 
$2^N \times 2^N$ 크기의 2차원 배열에서 특정 좌표 $(r, c)$가 몇 번째로 방문되는지를 찾아야 한다

탐색은 Z방향(좌상 $\to$ 우상 $\to$ 좌하 $\to$ 우하) 순서로 탐색한다.

## 풀이
Z 방향으로 움직일 때 우리는 한 가지 규칙을 알 수 있다.

바로 2의 제곱 단위로 사각형을 묶었을 때

왼쪽 위 -> 오른쪽 위 -> 왼쪽 아래 -> 오른쪽 아래 순으로 움직인다는 것이다.

그러므로 가장 작은 사각형의 단위, 즉 2x2 상태에서 부터 시작하여 

또 하나의 규칙을 적용하면 재귀를 완성 시킬 수 있다, 

바로 위, 이동 규칙대로 움직였을 때 좌표는

* Y, X
* Y, X + 현재 묶음으로 있는 사각형의 크기 (예: 2x2 사각형이면 1)
* Y + 사각형의 크기, X
* Y + 사각형의 크기, X + 사각형의 크기

이런 식으로 전개된다 고로, 현재 탐색하는 사각형 범위 내에 목표 좌표 $(r, c)$가 없다면 해당 넓이만큼 정답에 더하고 건너뛰는(Skip) 방식으로 효율적인 탐색이 가능하다.
